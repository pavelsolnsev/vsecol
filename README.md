# Демо-фронтенд (портфолио)

Nuxt-приложение: каталог образовательных организаций. Тексты, контакты и SEO настроены как **нейтральное демо**, без привязки к продакшену.

## Запуск

1. Скопируйте `.env.example` в `.env.local` и укажите `NUXT_PUBLIC_API_URL` (URL вашего API). Без него прокси `/api/**` отключён, часть страниц не загрузит данные.
2. `yarn install` и `yarn dev`.

Глобально включены `noindex, nofollow` и `robots.txt` с `Disallow: /`, чтобы демо не индексировалось. Аналитика (GTM, Метрика и т.д.) подключается только при непустых переменных из `.env.example`.
